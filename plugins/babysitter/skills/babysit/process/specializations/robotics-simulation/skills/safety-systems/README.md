# Safety System Skill

Robot safety system design and validation for industrial and collaborative robots.

## Purpose

Provides expert capabilities for designing functional safety systems for robots, ensuring compliance with ISO 10218, ISO 13482, and ISO/TS 15066 standards.

## Key Features

- Safety-rated monitoring functions
- Speed and separation monitoring
- Emergency stop integration
- Safety zone configuration
- ISO 13849 PL calculations
- Compliance documentation

## When to Use

- Designing robot safety systems
- Implementing collaborative robot safety
- Configuring safety zones
- Performing PL calculations
- Preparing safety documentation
