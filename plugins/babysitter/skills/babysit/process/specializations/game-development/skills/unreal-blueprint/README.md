# Unreal Blueprint Skill

## Overview

The Unreal Blueprint skill enables visual scripting for rapid game development in Unreal Engine.

## Purpose

Blueprints provide accessible game logic creation. This skill enables:

- **Visual Scripting**: Node-based programming
- **Rapid Prototyping**: Quick iteration
- **Designer Tools**: Non-programmer access
- **Integration**: C++ blueprint exposure

## Use Cases

1. **Gameplay Logic**: Player actions, abilities
2. **UI Systems**: Widget blueprints
3. **Level Scripting**: Triggers, sequences
4. **Prototyping**: Quick iterations

## Processes That Use This Skill

- **Core Mechanics Prototyping** (`core-mechanics-prototyping.js`)
- **Game Engine Setup** (`game-engine-setup.js`)

## Related Skills

- `unreal-development`: Core Unreal development
- `unreal-cpp`: C++ integration
- `fsm`: State machine patterns
