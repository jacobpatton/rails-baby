# Unreal Control Rig Skill

## Overview

The Unreal Control Rig skill enables procedural animation with runtime rig modifications.

## Purpose

Control Rig provides runtime animation control. This skill enables:

- **IK Solvers**: Foot, hand, look-at
- **Procedural Animation**: Runtime adjustments
- **Rig Blueprints**: Visual rig programming
- **Integration**: Animation BP connection

## Use Cases

1. **Foot IK**: Ground adaptation
2. **Look-At**: Head/eye tracking
3. **Aim Systems**: Weapon aiming
4. **Procedural Motion**: Physics-driven

## Processes That Use This Skill

- **Animation Implementation** (`animation-implementation.js`)

## Related Skills

- `unreal-development`: Core Unreal development
- `procedural-animation`: Animation concepts
- `animation-rigging`: General rigging
