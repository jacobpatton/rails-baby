# Unreal Sequencer Skill

## Overview

The Unreal Sequencer skill enables cinematic creation and cutscene development in Unreal Engine.

## Purpose

Sequencer provides non-linear editing. This skill enables:

- **Cinematics**: Film-quality cutscenes
- **Camera Work**: Professional camera control
- **Animation**: Actor and object keyframing
- **Integration**: Gameplay triggers

## Use Cases

1. **Story Cutscenes**: Narrative sequences
2. **Game Trailers**: Marketing materials
3. **In-Game Events**: Triggered sequences
4. **Virtual Production**: Film tools

## Processes That Use This Skill

- **Cutscene Implementation** (`cutscene-implementation.js`)
- **Narrative Design** (`narrative-design.js`)

## Related Skills

- `unreal-development`: Core Unreal development
- `unreal-control-rig`: Animation control
- `unreal-cinemachine`: Camera (Unity equivalent)
