# Unreal Chaos Skill

## Overview

The Unreal Chaos skill enables physics-based destruction, vehicles, and soft body simulation.

## Purpose

Chaos provides advanced physics. This skill enables:

- **Destruction**: Fracturing and debris
- **Vehicles**: Realistic driving
- **Cloth**: Soft body simulation
- **Fields**: Force effects

## Use Cases

1. **Building Destruction**: Explosive demolition
2. **Vehicles**: Racing, simulation
3. **Character Cloth**: Capes, hair
4. **Environmental**: Physics puzzles

## Processes That Use This Skill

- **Gameplay Systems** (`gameplay-systems.js`)
- **VFX Implementation** (`vfx-implementation.js`)

## Related Skills

- `unreal-development`: Core Unreal development
- `destruction`: Destruction patterns
