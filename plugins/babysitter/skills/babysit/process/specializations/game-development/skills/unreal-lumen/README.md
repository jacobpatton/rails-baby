# Unreal Lumen Skill

## Overview

The Unreal Lumen skill enables dynamic global illumination and reflections in Unreal Engine 5.

## Purpose

Lumen provides realistic lighting. This skill enables:

- **Dynamic GI**: Real-time bounced light
- **Reflections**: Screen and ray traced
- **Emissive Lighting**: Material-based lights
- **Performance Scaling**: SW/HW options

## Use Cases

1. **Interior Lighting**: Indirect illumination
2. **Day/Night Cycles**: Dynamic sun
3. **Reflective Surfaces**: Mirrors, metal
4. **Stylized Lighting**: Emissive effects

- **Art Asset Pipeline** (`art-asset-pipeline.js`)
- **Performance Optimization** (`performance-optimization.js`)
