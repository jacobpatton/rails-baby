# Unreal GAS Skill

## Overview

The Unreal GAS skill enables complex gameplay systems using the Gameplay Ability System.

## Purpose

GAS provides RPG-style ability systems. This skill enables:

- **Abilities**: Complex player actions
- **Attributes**: Stats and modifiers
- **Effects**: Buffs, debuffs, damage
- **Integration**: Networked gameplay

## Use Cases

1. **Action RPGs**: Abilities and stats
2. **MOBAs**: Complex character kits
3. **MMOs**: Ability systems
4. **Combat Games**: Attack systems

## Processes That Use This Skill

- **Gameplay Systems** (`gameplay-systems.js`)
- **Combat Design** related processes

## Related Skills

- `unreal-development`: Core Unreal development
- `unreal-cpp`: C++ integration
- `unreal-networking`: Multiplayer support
