# Unreal Nanite Skill

## Overview

The Unreal Nanite skill enables virtualized geometry for film-quality assets without LOD.

## Purpose

Nanite removes triangle budgets. This skill provides:

- **Virtual Geometry**: Automatic LOD
- **High-Poly Assets**: Film quality in games
- **Streaming**: Efficient mesh loading
- **Integration**: Lumen compatibility

## Use Cases

1. **Hero Assets**: Main characters, vehicles
2. **Environment**: Dense worlds
3. **Scanned Assets**: Photogrammetry
4. **Architecture**: Detailed buildings

## Processes That Use This Skill

- **Art Asset Pipeline** (`art-asset-pipeline.js`)
- **Performance Optimization** (`performance-optimization.js`)

## Related Skills

- `unreal-development`: Core Unreal development
- `unreal-lumen`: Lighting integration
- `lod`: Traditional LOD concepts
