# Unreal Niagara Skill

## Overview

The Unreal Niagara skill enables creation of advanced visual effects using Unreal's modern VFX system.

## Purpose

Niagara provides powerful VFX tools. This skill enables:

- **GPU Particles**: Massive particle counts
- **Custom Modules**: Extensible behavior
- **Simulation**: Physics-based effects
- **Integration**: Gameplay-driven effects

## Use Cases

1. **Combat Effects**: Explosions, impacts
2. **Environmental**: Weather, atmosphere
3. **Magic/Sci-Fi**: Energy, portals
4. **Destruction**: Debris, sparks

## Processes That Use This Skill

- **VFX Implementation** (`vfx-implementation.js`)
- **Art Asset Pipeline** (`art-asset-pipeline.js`)

## Related Skills

- `unreal-development`: Core Unreal development
- `particle-systems`: General particle concepts
- `unreal-materials`: Material integration
