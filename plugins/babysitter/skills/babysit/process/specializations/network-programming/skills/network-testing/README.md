# Network Testing Skill

Comprehensive network testing, benchmarking, and performance validation skill.

## Overview

This skill provides specialized capabilities for testing network performance, conducting load tests, and validating protocol implementations. It covers bandwidth testing, latency analysis, and chaos engineering for networks.

## Key Features

- Bandwidth and throughput testing
- HTTP load testing and benchmarking
- Latency measurement and analysis
- Protocol conformance testing
- Chaos engineering network tests

## When to Use

- Benchmarking network performance
- Load testing servers and services
- Validating protocol implementations
- Testing under degraded conditions
- Generating test reports

## Dependencies

- iperf3
- wrk/k6/hey
- mtr
- tc (traffic control)
- hping3
