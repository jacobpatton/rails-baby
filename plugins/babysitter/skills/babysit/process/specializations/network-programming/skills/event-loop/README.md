# Event Loop Skill

Expert skill for high-performance event-driven I/O programming and optimization.

## Overview

This skill provides specialized capabilities for building high-performance event-driven network applications. It covers platform-specific multiplexers, async I/O frameworks, and performance optimization techniques.

## Key Features

- epoll/kqueue/IOCP configuration
- Event loop performance analysis
- libuv, Tokio, Boost.Asio code generation
- io_uring implementation
- C10K+ optimization techniques

## When to Use

- Building high-concurrency servers
- Optimizing event loop performance
- Debugging event handling issues
- Implementing async I/O patterns
- Profiling network applications

## Dependencies

- strace/perf
- libuv
- io_uring
- System-specific profilers
