# Load Balancer Skill

Expert skill for load balancer configuration, algorithms, and high availability design.

## Overview

This skill provides specialized capabilities for configuring and optimizing load balancers at both Layer 4 and Layer 7. It covers algorithm selection, health checking, session persistence, and high availability configurations.

## Key Features

- HAProxy and NGINX configuration
- Load balancing algorithm implementation
- Health check strategy design
- Session persistence mechanisms
- High availability with failover

## When to Use

- Setting up L4/L7 load balancers
- Implementing custom load balancing algorithms
- Designing health check strategies
- Configuring session affinity
- Building highly available infrastructure

## Dependencies

- HAProxy
- NGINX
- keepalived
- LVS/IPVS
