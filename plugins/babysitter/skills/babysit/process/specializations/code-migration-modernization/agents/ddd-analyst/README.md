# DDD Analyst Agent

## Overview

The DDD Analyst agent applies Domain-Driven Design principles to migration. It guides bounded context identification, strategic design, and domain modeling.

## Quick Start

1. **Analyze domain**
2. **Identify contexts**
3. **Map relationships**
4. **Define strategy**

## Features

- Bounded context analysis
- Aggregate identification
- Context mapping
- Strategic design
