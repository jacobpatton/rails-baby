# Data Architect Agent

## Overview

The Data Architect Agent designs data architecture for migrated systems. It handles data decomposition, consistency patterns, and ownership definition.

## Quick Start

1. **Analyze data landscape**
2. **Design decomposition**
3. **Select patterns**
4. **Define ownership**

## Features

- Data decomposition planning
- Consistency pattern selection
- CQRS/Event Sourcing design
- Data mesh architecture
