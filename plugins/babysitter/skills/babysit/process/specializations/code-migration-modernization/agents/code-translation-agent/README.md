# Code Translation Agent

## Overview

The Code Translation Agent converts code between languages. It handles syntax translation, library mapping, and applies target language idioms.

## Quick Start

1. **Analyze source code**
2. **Map libraries**
3. **Translate code**
4. **Verify and test**

## Features

- Syntax translation
- Library mapping
- Idiom conversion
- Test translation
