# OBT Design Optimizer

Skill ID: SK-DEA-020

## Purpose

Designs and optimizes One Big Table (OBT) patterns. Balances denormalization benefits with maintainability for analytical use cases.

## Key Features

- Column selection optimization
- Denormalization strategy
- Nested/repeated field design
- Clustering key selection
- Partition strategy
- Query pattern analysis

## Applicable Processes

- OBT Creation
- BI Dashboard Development
- Query Optimization
