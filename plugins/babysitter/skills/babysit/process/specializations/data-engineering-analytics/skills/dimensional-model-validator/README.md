# Dimensional Model Validator

Skill ID: SK-DEA-008

## Purpose

Validates dimensional models against Kimball methodology best practices. Ensures models conform to proven design patterns for analytical workloads.

## Key Features

- Star/snowflake schema validation
- Grain definition verification
- Surrogate key validation
- SCD type checking
- Conformed dimension analysis
- Bus matrix compliance

## Applicable Processes

- Dimensional Model Design
- Data Warehouse Setup
- OBT Creation
