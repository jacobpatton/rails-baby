# BI Semantic Layer Generator

Skill ID: SK-DEA-009

## Purpose

Generates semantic layer definitions for BI tools from dimensional models. Automates translation to platform-specific semantic definitions.

## Key Features

- LookML generation
- Tableau data model generation
- Power BI semantic models
- Cube.js schema generation
- dbt metrics integration
- Hierarchy generation
- Security filter generation

## Applicable Processes

- Metrics Layer
- BI Dashboard Development
- Data Warehouse Setup
