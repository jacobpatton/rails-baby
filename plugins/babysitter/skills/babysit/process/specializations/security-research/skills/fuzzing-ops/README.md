# Fuzzing Operations Skill

Comprehensive fuzzing operations with AFL++, libFuzzer, and OSS-Fuzz integration.

## Quick Reference

- **Slug**: `fuzzing-ops`
- **Category**: Vulnerability Discovery
- **ID**: SK-002

## Key Features

- AFL++ campaign configuration and execution
- Instrumented binary building
- Seed corpus management
- Crash triage and deduplication
- Coverage monitoring and reporting
- Multiple fuzzer support (AFL++, libFuzzer, honggfuzz)

## Primary Use Cases

1. Automated vulnerability discovery
2. Coverage-guided fuzzing
3. Regression testing for security
4. Harness development
