# GTM Strategy Skill

Go-to-market planning and execution capabilities for product launches.

## Purpose

This skill provides comprehensive go-to-market planning capabilities. It helps product teams plan and execute effective launches with proper messaging, channel strategy, and sales enablement.

## Key Capabilities

- **Launch Planning**: Create tiered launch plans with timelines and checklists
- **Messaging Framework**: Develop positioning, value propositions, and key messages
- **Channel Strategy**: Identify and plan optimal launch channels
- **Sales Enablement**: Generate battlecards, objection handling, and talk tracks
- **Success Metrics**: Define and track launch success metrics

## Process Integration

Integrates with:
- Product Launch GTM Process
- Competitive Analysis Process
- Beta Program Process
- Stakeholder Alignment Process

## Usage

Invoke this skill when planning product launches, developing go-to-market strategies, or creating sales enablement materials.
