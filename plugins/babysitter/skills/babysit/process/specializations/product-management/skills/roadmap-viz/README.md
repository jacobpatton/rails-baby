# Roadmap Visualization Skill

Generate roadmap visualizations and planning artifacts for product planning.

## Purpose

This skill creates clear, stakeholder-appropriate roadmap visualizations in various formats. It transforms roadmap data into visual representations suitable for different audiences and purposes.

## Key Capabilities

- **Multiple Views**: Now/Next/Later, timeline, Gantt, dependency graphs
- **Audience Targeting**: Executive, engineering, sales, and customer views
- **Format Flexibility**: Markdown, Mermaid, CSV, JSON, ASCII output
- **Dependency Tracking**: Visualize initiative dependencies and risks
- **Resource Views**: Show team allocation and capacity

## Process Integration

Integrates with:
- Quarterly Roadmap Process
- Product Vision and Strategy Process
- Stakeholder Alignment Process
- Product Council Review Process

## Usage

Invoke this skill when creating roadmap presentations, planning documents, or stakeholder communications that require visual roadmap representations.
