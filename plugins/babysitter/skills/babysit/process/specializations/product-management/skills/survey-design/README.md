# Survey Design Skill

Design and analyze surveys for product validation and user research.

## Purpose

This skill provides comprehensive survey design and analysis capabilities. It helps product teams create effective surveys that gather meaningful feedback and supports rigorous analysis of survey results.

## Key Capabilities

- **Survey Templates**: PMF (Sean Ellis), NPS, CSAT, feature validation, exit surveys
- **Question Engineering**: Unbiased questions, appropriate scales, skip logic
- **Analysis Framework**: Statistical confidence, segmentation, pattern identification
- **Actionable Insights**: Transform data into recommendations

## Process Integration

Integrates with:
- Product Market Fit Process
- Beta Program Process
- Customer Advisory Board Process
- JTBD Analysis Process

## Usage

Invoke this skill when designing feedback surveys, analyzing survey responses, or validating product hypotheses through user research.
