# Customer Feedback Aggregation Skill

Aggregate and analyze customer feedback from multiple sources for product insights.

## Purpose

This skill provides comprehensive customer feedback aggregation and analysis capabilities. It helps product teams synthesize voice-of-customer data from multiple sources into actionable product insights.

## Key Capabilities

- **Multi-Source Aggregation**: Support tickets, NPS, sales calls, app reviews, interviews
- **Theme Extraction**: Identify patterns and themes across feedback sources
- **Feature Request Ranking**: Prioritize requests by frequency and impact
- **Sentiment Analysis**: Track sentiment trends over time
- **Segmentation**: Analyze feedback by customer attributes

## Process Integration

Integrates with:
- JTBD Analysis Process
- Feature Definition PRD Process
- RICE Prioritization Process
- Customer Advisory Board Process

## Usage

Invoke this skill when synthesizing customer feedback, prioritizing feature requests, or understanding customer pain points across multiple data sources.
