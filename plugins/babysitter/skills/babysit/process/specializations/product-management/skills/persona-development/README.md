# Persona Development Skill

Create and maintain user personas from research data for product targeting.

## Purpose

This skill provides comprehensive persona development capabilities. It helps product teams create data-driven user personas that guide product decisions, feature prioritization, and marketing strategies.

## Key Capabilities

- **Persona Creation**: Generate personas from interviews, surveys, and analytics
- **Jobs Mapping**: Define jobs-to-be-done for each persona
- **Feature Mapping**: Map product features to persona needs and priorities
- **Market Sizing**: Estimate TAM/SAM/SOM per persona segment
- **Persona Management**: Update and version personas over time

## Process Integration

Integrates with:
- User Story Mapping Process
- JTBD Analysis Process
- Feature Definition PRD Process
- Product Launch GTM Process

## Usage

Invoke this skill when creating new personas, updating existing personas with research, or mapping features to persona segments.
