# Parser Generator Skill

Expert skill for parser generation and implementation using LL, LR, LALR, PEG, and Pratt parsing techniques.

## Quick Reference

- **Category**: Compiler Frontend
- **ID**: SK-002

## Key Capabilities

- Parser generation from grammar specifications (ANTLR, Bison, tree-sitter)
- Recursive descent parser implementation with predictive parsing
- Pratt parser implementation for expression handling
- LALR/GLR parse table generation
- PEG parser implementation with packrat memoization
- Grammar conflict handling (shift-reduce, reduce-reduce)
- CST to AST transformation

## When to Use

Use this skill when implementing:
- Syntax analysis for a programming language
- Expression parsers with complex precedence rules
- Error-recovering parsers for IDE integration
- Parser generators for domain-specific languages
